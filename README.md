# codealpha_tasks
TASK-1 : Basic Network Sniffer
This project is a Basic Network Sniffer developed to monitor and analyze network traffic in real time. The tool captures packets transmitted across a network and displays important information such as source IP address, destination IP address, protocol type, and packet details. The project helps users understand how data travels through networks and provides hands-on experience with packet analysis, network monitoring, and cybersecurity concepts. It serves as an educational tool for learning network protocols and traffic inspection techniques.

TASK-2 : Phishing Awareness Training
This project is a Phishing Awareness Training presentation designed to educate users about phishing attacks and cybersecurity best practices. The training explains how to identify phishing emails, detect fake websites, understand common social engineering tactics, and protect sensitive information from cybercriminals. It also includes real-world examples and a short interactive quiz to reinforce learning. The goal of this project is to improve cybersecurity awareness and help users recognize and avoid phishing threats.
